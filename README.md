Here is my Emacs configuration. I strives to keep it *simple* as much as possible but there are still self-made crutches. Mostly I do not do programming in Emacs anymore. Main purpose is to write notes using [Org Mode](https://orgmode.org/).

Directory structure:

-   loadable - files in this directory can be loaded by `(require 'name)`
-   conf.d - standalone configurations
-   local.d - local part of standalone configurations

I do not use [use-package](https://jwiegley.github.io/use-package/) yet but I am going to do so.
